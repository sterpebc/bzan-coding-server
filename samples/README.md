# Sample databases

Throwaway `.db` files for manually exercising the app's upload/browse feature
during local development. They are not fixtures for an automated test suite
and are not read by any application code path — nothing hardcodes their
names or location.

These files are gitignored (see `.gitignore`), so add or replace whatever
`.db` files are handy for your own testing; there's no need to keep this
folder in sync with anyone else's.
